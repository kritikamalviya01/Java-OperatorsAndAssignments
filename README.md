# Java-OperatorsAndAssignments

1. Unary Operator :
        a. Increment -> ++
                Pre Increment -> x = ++y
                Post Increment -> x = y++
        b. Decrement -> --
                Pre Decrement -> x = --y
                Post Decrement -> x = y--
2. Arithmatic Operator :  +, -, *, /, %
3. Only operator that is overloaded in java is "+"
        Eg: 10+20 = 30
        Eg: "xy" + "ab" = xyab
